WebGL Single Dimensional Heat Equation Visualization

Number of Fourier Series Coefficients:

100

Current Time:

0

This project is a visualization of the Fourier series solution to the single dimensional heat equation along a rod. The mesh is generated and shaded using WebGL.

Click and drag on the mesh to view from different angles.

Source code is available at:

https://github.com/samfoucart/samfoucart
